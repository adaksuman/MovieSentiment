Moviesentiment

Hollywood upcoming movie success prediction based on social sentiment

Download this project as a .zip file Download this project as a tar.gz file

Introduction

Sentiment based decisions are affected by innumerable influences and are subjected to frequent changes.The basic idea of the application is to analyze popularity of a Hollywood movie from different social media(topsy,alchemyapi,rottentomato,neilson api) by computing sentiment values from posts in those media. The popularity is measures by "sentiment value" of the movie computed from the related posts/comments/reviews obtained from social media.
